## Development

```sh
# Install Ruby and Node
brew install node ruby

# Get the repo
git clone git@github.com:WeAreROLI/WeAreROLI.github.io
cd WeAreROLI.github.io

# Install the deps
gem install bundler
bundle install

# Run the site locally
bundle exec jekyll serve
```
